# chatroom-demo

> A Vue.js project

# Online Demo

[chatroom-demo](http://moonypoet.com)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
